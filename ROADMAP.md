# Roadmap

This roadmap outlines planned milestones for quorum-ai. Items are grouped by
version to show progression from the current POC to long-term goals. This is a
living document and does not include specific dates.

## Legend

- Status: Complete, In Progress, Planned
- Commitment: Committed, Tentative
- Priority: P0 (high), P1 (medium), P2 (low)

## v1.0 (Current POC)

Committed items for the initial release.

| Status | Priority | Commitment | Item |
| --- | --- | --- | --- |
| Complete | P0 | Committed | Core CLI with analyze/plan/execute commands |
| Complete | P0 | Committed | JSON-based state persistence |
| Complete | P0 | Committed | Git worktree isolation |
| In Progress | P0 | Committed | Claude and Gemini adapters |
| Complete | P1 | Committed | Jaccard consensus algorithm |
| In Progress | P1 | Committed | Basic TUI with progress display |
